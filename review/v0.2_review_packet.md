## G. Remaining limitations (non-claims)

- W3C VC compliance is NOT claimed: no JSON-LD context processing, no URDNA2015 RDF normalization. Label is "VC-inspired signed credential."
- Enterprise PKI / HSM-grade key isolation is NOT claimed.
- Distributed revocation is NOT claimed; the registry is per-process.
- Arbitrary untrusted-code WASM safety is NOT claimed; the sandbox is verified for the calculator and two adversarial fixtures only.
- Wall-clock timeout on WASM is NOT claimed; fuel bounds compute, not seconds.
- Cryptographic proof from the TLA+ model is NOT claimed; only safety, only against bounded constants.
- Liveness verification is NOT claimed.
- Side-channel resistance (timing, cache, Spectre-class) is NOT claimed.
- Production deployment readiness is NOT claimed.
- Multi-tenant sandbox isolation is NOT claimed; one trust domain at a time.
- Interoperability with MCP, A2A, ACP, ANP ecosystems beyond protocol-shape similarity is NOT claimed.
