# Limitations

SIFR v0.1 is not production-ready.

Limitations:
- Local transport only.
- No replay cache.
- No grant revocation.
- No public-key infrastructure.
- DID-style strings only; no DID resolution.
- No Verifiable Credentials.
- No QUIC implementation.
- No WASM/WASI sandbox.
- No formal security proof.
- TensorFrame is demo encoding, not real KV-cache sharing.
- Benchmarks are local microbenchmarks.
